What NetSentry Is

NetSentry is a practical cybersecurity gateway project built to turn a Debian machine into a controlled Wi-Fi gateway, DNS filtering point, firewall, router, IDS visibility point, log collector, and web monitoring interface.

Motivation

The goal is to build a real defensive network appliance, not only run isolated tools. NetSentry connects access control, packet flow, DNS filtering, firewall policy, IDS alerts, and logs into one explainable system.

What We Want to Achieve

The project aims to prove that a low-cost Debian server can become a monitored gateway for client devices, with a clean admin interface and a clear security boundary between public pages and protected operator data.

Current Direction

V1 focuses on visibility: public project pages, admin status, clients, DNS, firewall, IDS, logs, and network views. Later versions can add controlled response through a privileged agent, not direct Flask root commands.
